Linux Basic で紹介しなかったものについて解説をします。

3.1. cp¶

ファイルをコピーするコマンド

3.1.1. 文法¶

cp コピー元ファイル　コピー後名

3.1.2. test1.dat をtest2.datという名前でコピーする。cp test1.dat test2.dat¶

$ ls
text1.dat

$ cp text1.dat test2.dat # test1.dat をtest2.datという名前でコピーする。

$ ls
text1.dat test2.dat

3.1.3. fileディレクトリの中のtext1.datを一つ上のディレクトリにコピーしてくる。cp file/text1.dat ./¶

$ ls 
file

$ cp file/text1.dat ./

$ ls
file text1.dat

3.1.4. fileディレクトリをfile_sameという名前でコピーする。cp -r file file_same¶

$ ls
file text1.dat

$ cp -r file file_same

$ ls 
file file_same text1.dat

このように、ファイルならcpで問題ないがディレクトリをコピーする場合、cp -rとする。

3.2. mv¶

  • ファイル名を変更するコマンド

  • ファイルを移動するコマンド

3.2.1. ファイル名変更時の文法¶

mv 変更前ファイル名　変更後ファイル名

3.2.2. text1.dat をtext_new.datに改名mv text1.dat text_new.dat¶

$ ls
text1.dat
$ mv text1.dat text_new.dat

$ ls
text_new.dat

3.2.3. ファイル移動時の文法¶

mv 移動したいファイル　移動先のディレクトリ

3.2.3.1. text1.dat をfileの中に入れるmv text1.dat file¶

$ ls
file text1.dat

$ mv test1.dat file

$ ls
file

$ ls file/
test1.dat

移動先のディレクトリはすでに存在している必要がある。まだ作ってなければtext1.datがfileに改名されてしまう。

3.3. rm¶

ファイルを削除するコマンド

3.3.1. 文法¶

rm 削除したいファイル

3.3.2. ファイル test.dat を削除rm test.dat¶

$ ls
test.dat

$ rm test.dat

$ ls

このように、確認メッセージなどがなく、削除されるため注意が必要！！

（ゴミ箱に行ったりもせずいきなり消える）

3.3.3. ディレクトリ file/ を削除 rm -r file¶

$ ls
file 

$ rm -r file

$ ls

cp のと同様にディレクトリに対して行うには-rのオプションが必要。

3.4. echo¶

文字の表示をするコマンド

3.4.1. 文法¶

echo 表示したい文字

3.4.2. ¶

3.4.3. I am a geniusと表示する。echo "I am a genius"¶

$ echo "I am a genius"
I am a genius

これだけだと何の面白みもないが、あとで紹介する「リダイレクト」を活用することで利便性が格段に上がる。

3.4.4. I am a geniusという内容が書かれたtest.datというファイルを作成。echo "I am a genius" > test.dat¶

$ echo "I am a genius" > test.dat
$ cat test.dat
I am a genius

cat はファイルの中身を表示するコマンドで、次に説明する。

3.5. cat¶

ファイルの中身を表示する。

3.5.1. 文法¶

cat 中身を表示したいファイル

3.5.2. I am a geniusという内容が書かれたtest.datというファイルの中身を見るcat test.dat¶

$ cat test.dat
I am a genius

3.5.3. ~/.bashrcの中身を表示してみる。¶
